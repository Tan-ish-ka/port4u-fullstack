# 🌐 Port4U – Portfolio Management Web Application

**Port4U** is a role-based web application developed as a **team project**, where users can easily create and publish their portfolios by selecting templates and entering their details.

👉 **My Contribution:** Backend development, API integration, authentication system, and deployment.

---

* ⚙️ **Backend (Render):**
  https://port4u-backend-jgg8.onrender.com

---

## 🚀 Features

* 🔐 Email & password-based authentication
* ✉️ OTP verification during registration
* 🧑‍💻 Role-based dashboards (Client & Admin)
* 📄 Dynamic portfolio generation
* 🗃️ Database integration using Spring Data JPA
* 🌐 REST API architecture
* 🎨 Simple frontend using HTML, CSS, JS

---

## 🛠️ Tech Stack

| Layer      | Technology                           |
| ---------- | ------------------------------------ |
| Backend    | Spring Boot, Spring Security         |
| Frontend   | HTML, CSS, JavaScript                |
| Database   | H2 (in-memory)                       |
| ORM        | Spring Data JPA                      |
| Deployment | Render (Backend), Netlify (Frontend) |

## ⚙️ Running Locally

### Prerequisites

* Java 17+
* Maven installed

### Steps

```bash
git clone https://github.com/Tan-ish-ka/port4u-fullstack.git
cd PORT4U-FINAL
mvn spring-boot:run
```

Then open:

```
http://port4u-backend-jgg8.onrender.com/signup.html
```
